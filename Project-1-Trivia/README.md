
# Trivia Quiz for Geography in Bulgaria
Trivia Quiz for Geography in Bulgaria is a simple educational game developed in Python with a graphical user interface (GUI).
The game helps users test and improve their knowledge about the geography of Bulgaria in an interactive way.
<img alt="Image" width="200px" src="https://en.wikipedia.org/wiki/Geography_of_Bulgaria"></img>
Players are presented with multiple-choice questions and must choose the correct answer. 
At the end of the quiz, the user receives a final score.
Used techonology:
Python
* tkinter /for GUI/
* random
